Это гайд по установке моего ВПН

Для начала необходимо скачать приложение v2RayTun на свой телефон через Google Play/App Store (В зависимости от того, какой у вас телефон)

![photo_2026-04-07_20-11-27](https://github.com/user-attachments/assets/c5a0ca65-ce3b-474c-8104-bd9dd9e57f87)

(на месте стрелочки будет кнопка "Установить")

Далее копируем представленную ниже ссылку в буфер обмена (Если вы не знаете, как это сделать, чуть ниже это подробно расписано)
https://weirdmsk.mikata.ru:1984/sub/mskswe/2osum6540brtyzt9

Удерживая палец на тексте ссылки выскочит небольшое окошко (такое же, как на фото, или похожее), в котором нужно выбрать один из выделенных пунктов (или похожих, в них нет особой разницы)

![photo_2026-04-07_20-18-36](https://github.com/user-attachments/assets/daff58e0-1b55-45a9-89f9-7badbcab6295)

После того, как мы успешно скопировали ссылку, нужно вставить её в установленный ранее клиент. Для этого сначала заходим в приложение, которое мы недавно скачали

После входа нам нужно нажать на плюсик справа сверху, после чего в выскочившем окошке выбираем пункт "Импорт из буфера обмена"

![photo_2026-04-07_20-26-24](https://github.com/user-attachments/assets/a8a30d6d-d86f-4a44-8519-d58cddc4590a) 
![photo_2026-04-07_20-26-28](https://github.com/user-attachments/assets/ce9ea9d0-f601-42b3-b8e4-dfa24a157f15)

После того, как все нужные действия были сделаны, можно увидеть, что нужная конфигурация добавилась в приложение. На этом вся найстройка завершена

Для использования ВПН просто нажимаем кнопку включения посередине
